# Questionnaires et gestion de l'information
Le but des questionnaires étaient de récupérer les informations de l'utilisateur pour fournir une expérience utilisateur unique.

Il nous fallait donc: 
* connaître les habitudes de l'utilisateur
* recueuillir les données de disponibilités de l'utilisateur
* connaître le niveau de l'utilisateur

# Globalisation des données
Pour créer une expérience qui puisse évoluer avec le temps et l'investissement des équipes, nous sommes partis sur une organisation sous forme de base de données codée en XML.

Cette base de données nous permettait d'accéder à certaines données nécessaires à la réalisation d'une expérience utilisateur dynamique.

Celle-ci contient entre autre:
* plusieurs sports
* plusieurs muscles et groupes musculaires
* plusieurs exercices ou mouvements


# Utilisation du questionnaire

Une fois le questionnaire remplie il a fallut trier les exercices qui convenaient à l'utilisateur en fonction de ses choix, de son matérielle et du temps disponible.
